En la etapa de pruebas identificamos los errores de funcionamiento permitiendo que el sistema sea entregado como un producto de alta calidad al cliente.

 

Las pruebas de Unidad: verificar que las funcionalidades implementadas presenten limitaciones, podemos realizar test para analizar el antes y después de las funcionalidades desarrolladas.

 

Pruebas de Integración: Se realizan para verificar el funcionamiento del sistema cuando juntamos los componentes y garantizar que el sistema pueda seguir su avance.

 

Pruebas que se realizan desde el perfil del usuario final permitiendo realizar ajustes de la interfaz del usuario funcional.

 

Pruebas beta se realizan al producto que no es hecho a la medida, esta es aplicada por los usuarios finales logrando que el producto tenga éxito en el mercado.

 

Pruebas de aceptación, entrega del producto final del desarrollo y el comienzo de la etapa de mantenimiento.
