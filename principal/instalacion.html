Instalación/Despliegue

Finalizadas las etapas de desarrollo ponemos en funcionamiento el sistema, para la instalación debemos contemplar el hardware y software (actualizados para evitar problemas de seguridad) y lograr mitigar problemas de compatibilidad.

 

Realizar la planificación de transición del sistema antiguo al nuevo permite no generar interrupciones en las funcionalidades del sistema.
