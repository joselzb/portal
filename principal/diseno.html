Diseño

En esta etapa los modelos de diseño nos permiten caracterizar el sistema (funcionalidades), de una forma más efectiva identificando la trazabilidad de los requerimientos.

 

Arquitectura multicapa:

 

En la actualidad los desarrollos de software se realizan en un sistema de tres capas.

 

·         Capa de presentación interfaz creada a la presentación del usuario.

·         Lógica de la aplicación implementada para trabajar sobre un módulo orientado a objetos.

·         Acceso a base de datos donde se realiza toda la gestión de base de datos.

 

Trabajar en estas tres capas permite realizar un trabajo colaborativo logrando que es el sistema sea portable y minimizando la creación de código, de igual forma hacer un mejor mantenimiento e implementación de nuevas funcionalidades.
